      Selenium

      Описание

      Selenium WebDriver — это инструмент для автоматизации действий веб-браузера. В большинстве случаев используется для тестирования Web-приложений, но этим не ограничивается. В частности, он может быть использован для решения рутинных задач администрирования сайта или регулярного получения данных из различных источников. Википедия
      Дата выхода: 30 апреля 2002 г.
      Последняя версия: 4.5.0 (28 сентября 2022; 6 месяцев назад)
      Лицензия: Apache License 2.0
      Написана на: Java
      Операционная система: Microsoft Windows, GNU/Linux, Apple Mac OS
      Тип: тестирование программного обеспечения
      Языки программирования: Python, Java, JavaScript, C Sharp, Ruby
